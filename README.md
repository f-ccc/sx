# 校园选课记录检索与大数据分析系统

## 📋 项目简介
数据结构与算法课程设计项目 —— 基于C语言（C89标准）开发的选课记录管理系统，包含三种数据结构（双向链表、AVL树、哈希表）的实现与性能对比分析。

## 🛠 编译方法
```bash
# Dev-C++ v5.9.2 直接打开项目编译
# 或命令行：
gcc -std=c89 -pedantic -Wall -Wextra -o main.exe main.c record.c csv.c list.c avl.c hash.c sort.c stat.c performance.c
```

## ✅ 功能清单
| 任务 | 功能 | 状态 |
|:----:|------|:----:|
| 1 | 数据结构设计与基本操作（增删改查） | ✅ |
| 2 | CSV数据持久化 | ✅ |

## 📁 项目结构
```
├── main.c           # 主程序（CLI菜单交互）
├── record.h/c       # 选课记录定义、数据生成
├── list.h/c         # 双向链表
├── avl.h/c          # AVL树（自平衡二叉搜索树）
├── hash.h/c         # 哈希表（链地址法）
├── csv.h/c          # CSV文件读写
├── sort.h/c         # 筛选与排序
├── stat.h/c         # 统计分析
├── performance.h/c  # 性能测试与复杂度验证
├── test.c           # 自动化测试程序
└── data/            # 数据文件目录
```
